🤖 Programación por bloques en Arduino

Arduino es una plataforma electrónica que permite controlar dispositivos físicos como luces, motores y sensores mediante programación.

En lugar de escribir código complejo, se puede usar la programación por bloques, donde se utilizan piezas visuales que representan instrucciones.

🧠 ¿Qué es programar?

Programar es darle instrucciones a un dispositivo para que realice una acción.

👉 Ejemplo:
Si presiono un botón → encender LED

---

🧩 Tipos de bloques

🔹 Bloque de inicio

Indica cuándo comienza el programa.

👉 Es el punto de partida de todas las instrucciones.

---

🔹 Bloques de acción

Son los que ejecutan acciones en el sistema.

  • Encender LED
  • Apagar motor
  • Emitir sonido
---

🔹 Bloques de control (condicionales)

Permiten tomar decisiones según una condición.

👉 Ejemplo:
Si hay movimiento → encender LED
Si no → apagar LED

---

🔹 Bloques de repetición (bucles)

Repiten una acción varias veces o de forma continua.

  • Repetir siempre
  • Repetir cada cierto tiempo

👉 Ejemplo: hacer parpadear un LED cada segundo.

---

🔹 Bloques de sensores

Permiten leer información del entorno.

  • Sensor de movimiento
  • Sensor de luz
  • Sensor de temperatura

👉 Estos bloques permiten que el sistema reaccione al ambiente.

---

🎯 Ejemplo completo

Un programa puede funcionar así:

  • Leer sensor
  • Evaluar condición
  • Ejecutar acción
  • Repetir proceso

👉 Esto permite automatizar tareas como encender luces automáticamente.

---

🚀 Importancia

La programación por bloques facilita el aprendizaje porque:

  • Es visual y fácil de entender
  • Evita errores de sintaxis
  • Permite aprender lógica de programación
---

🎯 Conclusión

Arduino y la programación por bloques permiten controlar dispositivos electrónicos de forma sencilla, ayudando a comprender cómo funcionan los sistemas automatizados.

🤖 ¿Qué hará el programa?

Si el sensor detecta movimiento → encender LED
