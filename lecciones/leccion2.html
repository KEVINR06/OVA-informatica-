Contenido

Un sistema de numeración es una forma de representar cantidades mediante símbolos. El sistema más usado es el sistema decimal.

Los computadores utilizan el sistema binario porque trabajan con señales eléctricas que pueden estar encendidas o apagadas.

  • Decimal → base 10
  • Binario → base 2
  • Octal → base 8
  • Hexadecimal → base 16

Ejemplo de conversión: 1010 (binario) = 1×8 + 0×4 + 1×2 + 0×1 = 10 decimal
