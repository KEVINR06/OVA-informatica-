📘 Contenido

El pseudocódigo es una forma de escribir instrucciones de manera sencilla para resolver un problema, sin usar un lenguaje de programación específico.

Se utiliza para planear algoritmos antes de programarlos, facilitando la comprensión de la lógica del problema.

Un algoritmo es una secuencia de pasos ordenados para resolver una tarea o problema.

Ejemplo de pseudocódigo

  • Inicio
  • Leer número
  • Si número > 0 entonces
  • Mostrar "Es positivo"
  • Fin si
  • Fin

El pseudocódigo utiliza palabras clave como: Inicio, Fin, Si, Entonces, Mientras, Para, etc.
