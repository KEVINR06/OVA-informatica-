📘 Contenido

La informática es la ciencia que se encarga de estudiar cómo se procesa, almacena y transmite la información utilizando dispositivos electrónicos como los computadores.

Hoy en día, la informática está presente en casi todo lo que hacemos: cuando usamos un celular, navegamos en internet, jugamos videojuegos o realizamos tareas escolares.

Un sistema informático está compuesto por hardware (partes físicas) y software (programas). Ambos trabajan juntos para que el computador funcione correctamente.

Hardware

  • Teclado
  • Mouse
  • Pantalla
  • Procesador
  • Memoria RAM

Software

  • Sistema operativo
  • Aplicaciones
  • Videojuegos
