📘 Contenido

HTML (HyperText Markup Language) es el lenguaje que se utiliza para crear la estructura de una página web.

CSS (Cascading Style Sheets) se utiliza para dar estilo a esa estructura, como colores, tamaños y diseño.

HTML organiza el contenido y CSS lo hace visualmente atractivo.

Ejemplo básico HTML

  • <h1>Título</h1>
  • <p>Párrafo</p>

Ejemplo básico CSS

  • body { background: black; }
  • h1 { color: white; }
