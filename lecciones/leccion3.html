📘 ¿Qué es HTML?

HTML significa HyperText Markup Language. Es el lenguaje que se utiliza para crear páginas web.

HTML funciona mediante etiquetas que indican al navegador cómo mostrar la información.

Ejemplo de etiquetas:


<h1>Título</h1>

<p>Este es un párrafo</p>

Las etiquetas siempre tienen una apertura y un cierre.

¿Qué es CSS?

CSS significa Cascading Style Sheets.

Sirve para darle estilo a las páginas web, como colores, tamaños de letra, fondos y organización visual.

Ejemplo:


h1{
color:blue;
font-size:30px;
}

HTML crea la estructura de la página CSS define cómo se ve esa página.
