📘 Contenido

HTML (HyperText Markup Language) es el lenguaje que se usa para crear la estructura de una página web.

CSS (Cascading Style Sheets) se utiliza para dar estilo, como colores, tamaños y diseño.

HTML define el contenido, mientras CSS define cómo se ve ese contenido.

Ejemplo HTML

  • <h1>Hola</h1>
  • <p>Texto</p>

Ejemplo CSS

  • h1 { color: blue; }
  • p { font-size: 20px; }
