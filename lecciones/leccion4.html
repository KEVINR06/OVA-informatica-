🧠 Algoritmos y programación en Arduino

Un algoritmo es un conjunto de pasos ordenados que se siguen para resolver un problema.

👉 Ejemplo:

  • Encender sistema
  • Leer sensor
  • Tomar decisión
  • Ejecutar acción
---

💻 Estructura de un programa en Arduino

🔹 void setup()

Es una función que se ejecuta una sola vez al iniciar el programa.

Se utiliza para:

  • Configurar pines
  • Inicializar variables
  • Preparar el sistema

👉 Es como preparar todo antes de empezar.

---

🔹 void loop()

Es una función que se ejecuta de forma repetitiva mientras el programa está activo.

Se utiliza para:

  • Leer sensores
  • Tomar decisiones
  • Controlar dispositivos

👉 Es el corazón del programa.

---

🔁 Ejemplo de funcionamiento

void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}

👉 Este programa hace que un LED parpadee.

---

🔷 Diagramas de flujo

Un diagrama de flujo es una representación visual de un algoritmo.

---

🔵 Óvalo (Inicio/Fin)

Indica dónde comienza o termina el proceso.

---

🔷 Rectángulo (Proceso)

Representa una acción o instrucción.

👉 Ejemplo: Encender LED

---

🔶 Rombo (Decisión)

Evalúa una condición.

👉 Ejemplo: ¿Hay movimiento?

Tiene dos caminos:

  • Sí
  • No
---

➡️ Flechas

Indican la dirección del flujo del programa.

---

🎯 Ejemplo de diagrama lógico

  • Inicio
  • Leer sensor
  • ¿Hay movimiento?
  • Sí → Encender LED
  • No → Apagar LED
  • Repetir
---

🚀 Importancia

Los algoritmos y diagramas de flujo ayudan a:

  • Organizar ideas
  • Evitar errores
  • Comprender mejor los programas
---

🎯 Conclusión

Comprender el uso de setup(), loop() y los diagramas de flujo permite desarrollar programas eficientes y entender el funcionamiento de sistemas electrónicos y automatizados.
