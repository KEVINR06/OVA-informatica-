📘 Introducción a la informática

La informática es la ciencia que se encarga del procesamiento automático de la información mediante dispositivos electrónicos como computadores, celulares y sistemas digitales.

Su objetivo es permitir que la información sea recibida, almacenada, procesada y mostrada de forma rápida y eficiente.

🧩 Componentes de un sistema informático

💻 Hardware

El hardware corresponde a todas las partes físicas del computador, es decir, todo lo que se puede ver y tocar.

  • Teclado: Permite ingresar datos
  • Mouse: Controla el cursor
  • Monitor: Muestra la información
  • CPU: Procesa los datos
  • Memoria RAM: Almacena información temporal

👉 El hardware necesita instrucciones para funcionar.

🧠 Software

El software son los programas que le indican al hardware qué hacer.

  • Sistema operativo (Windows, Linux)
  • Aplicaciones (Word, Excel)
  • Videojuegos

👉 Sin software, el hardware no tiene utilidad.

⚖️ Diferencia entre hardware y software

Hardware Software
Es físico Es digital
Se puede tocar No se puede tocar
Ej: teclado Ej: Word
Ejecuta acciones Da instrucciones

🎯 Conclusión

El hardware y el software trabajan juntos para permitir el funcionamiento de los dispositivos tecnológicos y facilitar el uso de la informática en la vida diaria.
